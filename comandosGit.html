Comandos do GIT

  • git add . Preprara os arqivos para serem enviados.
  • git commit -m "mensagem" Adiciona uma mensagem que descreve o que foi alterado.
  • git push Envia os dados para o github.
  • git pull Traz os dados do github para usa maquina
  • git clone clona um projeto e disponibiliza na sua maquina no local.
